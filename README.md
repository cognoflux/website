### Prerequisites

- Node.js 18+ 
- npm, yarn, pnpm, or bun

### Installation

```bash
# Clone the repository
git clone git@github.com:cognoflux/website.git
cd website

# Install dependencies
npm install

# Run development server
npm run dev
```

### Build for Production

